# Brutalist

Brutalist design communicates confidence through simplicity, boldness, and unapologetic structure.

It embraces honesty over polish and clarity over refinement.

Every element should feel direct and intentional.

---

# Core Principles

- Honesty over decoration
- Bold typography
- Strong visual hierarchy
- High contrast
- Functional confidence

---

# Typography

Typography should dominate the visual identity.

Large headlines and strong weight create emphasis.

---

# Layout

Layouts should embrace visible structure.

Grids, alignment, and spacing should remain obvious rather than hidden.

---

# Color Philosophy

Color should support hierarchy through contrast.

Avoid decorative palettes.

---

# Shapes

Sharp edges.

Simple geometry.

Minimal ornamentation.

---

# Spacing

Spacing should create structure rather than elegance.

Avoid excessive whitespace.

---

# Motion

Motion should remain minimal.

Interactions should feel immediate.

---

# Imagery

Authentic imagery.

Minimal visual manipulation.

Graphics should reinforce communication rather than decoration.

---

# Components

Components should communicate purpose immediately.

Decoration should never obscure functionality.

---

# Suitable Architectures

- Portfolio
- Creative Agencies
- Marketing
- Community

---

# Avoid

Avoid unnecessary polish, soft aesthetics, decorative gradients, or excessive animation.
