# Documentation SEO

Documentation websites earn visibility by helping users successfully understand and use a product or platform.

SEO should prioritize clarity, discoverability, and long-term usefulness over marketing language.

Documentation should become the definitive reference.

---

# Search Intent

Visitors typically search to:

- Learn APIs
- Solve implementation problems
- Read guides
- Find examples
- Troubleshoot errors

Documentation should progress naturally from beginner to advanced topics.
